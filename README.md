# Teachers Tools

A small personal repo where I build small tools that help teachers. Each tool is born from noticing a little thing that would be helpful in the classroom—and may grow into something more over time.

## Tools

### Word Tracing Practice

A web-based printable practice sheet for kindergarten (ages 3–4) learning to write. Kids trace words over dotted letters, with a days-of-the-week table for repeated practice.

[Open the tool →](word-tracing/index.html)
